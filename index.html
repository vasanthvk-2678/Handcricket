Hand Cricket

This is a game played with bare hands like Rock Paper Scissors. I think Most of u are famliar with this game. I used to play this with my friends when I get bored in school. Try to play with ur friends using ur hands.It's very fun playing it.
Rules:
  1. Toss: To select What you opt for i.e. batting or bowling...That is Heads or Tails
    For ex: It will ask u to select heads or tails...
    If u win the toss, u can choose whatever u want...
    Or else Computer chooses to either BAT or BOWL

  2. Batting: It will ask u for a number from 1 to 6...
    If u chose number x and computer chose any number otherthan x .Then it means, u have scored x runs...
    If the computer choose the same number u chose.
    You are declared to be out and the batting turn goes for the computer...

  3. Bowling: It is same as doing batting but here you are bowler..
    You should guess what number the computer chooses. So that You can out him..

  4. Result: If you scored more runs than the computer...It means You won the match...
    And the same applies to the computer
